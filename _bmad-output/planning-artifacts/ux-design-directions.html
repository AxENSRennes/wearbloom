Wearbloom

Design Directions

Four visual explorations for the Wearbloom mobile experience. Each direction shares the same premium foundation — white backgrounds, near-black type, nude-blush accents — but takes a distinct approach to layout, density, and interaction style. Each direction shows three core screens: Wardrobe Grid, Garment Detail, and Try-On Render Result.

Direction 01

Editorial Clean

Glossier meets Vogue

Maximum whitespace with a magazine-editorial sensibility. Large serif headlines, generous spacing, and a two-column grid with breathing room let each garment feel considered. The detail view centers the garment with a full-width CTA, while the render result goes full-bleed with minimal chrome.

Direction 02

Compact Browse

Pinterest density meets luxury fashion

A denser three-column grid puts more garments on screen at once, perfect for larger wardrobes. Category pills are prominent and sticky. The garment detail uses a bottom-sheet pattern, keeping context visible behind. The render result is full-screen with floating action buttons.

Direction 03

Immersive Visual

Instagram Stories meets fashion catalog

Edge-to-edge imagery with minimal chrome. The two-column grid drops visible borders entirely — garment photos float without card containers. The detail view expands imagery to fill the screen with an overlaid CTA. The render result is completely full-screen, dismissed with a swipe.

Direction 04

Card-Based Refined

Apple design language meets fashion

Elevated card design with subtle shadows and rounded corners create a structured, organized feel with clear visual hierarchy. Each garment card carries a small category badge. The detail view uses a card-style modal, and the render result stays within a card with action buttons below.
